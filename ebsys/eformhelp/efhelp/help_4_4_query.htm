1.查询视图
    查询视图是在grid控件上显示多条记录的查询结果,grid控件上的记录不能修改,如要在grid控件上能删除记录的话，需要设置保存表名和主键,能对数据进行分页显示和设置每页显示多少条记录.可以在查询视图上通过点击按钮或超链接的方式打开查看视图,编辑视图,新增视图等等.
2.不同设置的查询视图
2.1 不带条件的查询视图
    单击 不带条件的查询视图 进入不带条件的查询视图设计界面.在表单上添加一个数据集控件和一条表格控件.双击表单中的数据集控件,在弹出的数据集属性的打开页中,输入要查询的sql语句,字段设置为只读.

    设置保存表名为FCS_EMPLOYEE,提交类型,智能提交变动过的记录.这是因为删除记录时需要这些记录.

    在表单属性窗口的事件页中的打开之后输入：

fcpubdata.viewWinSee='help_4_4_query1';fcpubdata.viewWinAdd='help_4_4_query2';fcpubdata.viewWinEdit='help_4_4_query3';

    fcpubdata.viewWinSee是查看视图表单的djsn,fcpubdata.viewWinAdd编辑视图表单的djsn,fcpubdata.viewWinEdit编辑视图表单的djsn.

    在表单属性窗口中设置表单的工具栏为自定义工具栏,单击【设置】按钮进入自定义工具栏页面,单击【添加】按钮,增加行,单击行中的单元格,单击查看视图，就选择了查看视图按钮,如下图设置自己需要的按钮.

function uf_ok(){//单击确定按钮返回选择的查询条件

         var sRet = '';

         sRet = new Eapi.Str().trim( uf_where());

         if(sRet == ""){

                   alert("生成的SQL语句为空!");

                   return;

         }

         window.returnValue = [sRet];

}

function uf_where(){//拼接条件语句

         var l = grid2.Rows;

         var sWhere = "";

         var zifu="";

         for(var i=1;i<l;i++){

                   var fdtype = grid2.tab.rows(i).cells(8).innerText;

                  

                   var fdName = grid2.tab.rows(i).cells(7).innerText;

                   if(fdtype == "字符") zifu="'";

                   var sOr = "";

                   if(i<l-1) {

                            if(grid2.tab.rows(i).cells(6).innerText == "或者"){

                                     sOr =  " or ";

                            }else  {

                                     sOr = " and ";

                            }

                   }

                   sWhere += grid2.tab.rows(i).cells(1).innerText + fdName + _OpNameTo(grid2.tab.rows(i).cells(3).innerText)+ zifu + grid2.tab.rows(i).cells(4).innerText + zifu +grid2.tab.rows(i).cells(5).innerText + sOr;

         }

         sWhere = new Eapi.Str().trim(sWhere);

         return sWhere;

}

 

function _OpNameTo(sName) {

         var sRet1 = "";

         switch (sName) {

                  

                   case "＝" : sRet1 = "="; break;

                   case "＞" : sRet1 = ">"; break;

                   case "＜" : sRet1 = "<"; break;

                   case "＞＝" : sRet1 = ">="; break;

                   case "！＝" : sRet1 = "!="; break;

                   case "开始于" : sRet1 = " like "; break;

         }

         return sRet1;

}

3.与其它视图对比
    查询视图是表格控件显示多条只读记录,单击【查看】【添加】【编辑】按钮在当前页弹出相关的查看视图表单、新增视图表单、编辑视图表单.查询视图主要是根据条件生成数据集的SQL语句。
