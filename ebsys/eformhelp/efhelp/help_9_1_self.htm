1.添加或引用后所有表单都能使用
1.1 给fcpub.js中的fcpubdata的cssFiles属性赋值引入自己的css文件
       可以在fcpub.js的fcpubdata的cssFiles属性中引入,如果引入多个样式文件用逗号隔开,在这引入的css文件,eform中的表单都能调用到它的样式.如下图
1.2在dj.css文件中添加css样式
       在ebsys/fceform/css/dj.css样式文件中添加你的css样式,eform中的所有表单都能应用到dj.css文件的样式,所以你在dj.css中添加的样式,所有表单都能引用到.
2. 在表单的附加页面中定义样式只有当前表单能用
       在表单的附加页面中定义的样式只有当前表单能用,如果双击控件,打开控件的属性窗口,选择设置了字体大小,颜色后,附加页面的的样式将不起作用,因为引用类名(class)没有在控件的属性窗口中选择产生的style="font-size:12px"优先级高.定义样式界面如下:<br>

      在控件属性页中可以直接输入类名,来设置控件的样式,或者在属性中定义类名.如下图
3. 引入自己的js文件
        增加函数方式如下:
       1 此函数只是当前表单用,可以直接在此表单的自定义函数中写即可.
       2 此函数是所有的表单共用.因为默认情况下所有的表单都引用了 fceformext/js/userfunc.js 文件. 所以可以将函数加到 fceformext/js/userfunc.js 文件中.
       3 在fceformext/js/userfunc.js文件中直接引入自己的一个js文件.

// load some scripts (function() { var scripts = [ fcpubdata.Path + "/fceform/js/fcvalid.js" ]; var head = document.getElementsByTagName("head")[0]; for (var i = 0; i < scripts.length; ++i) { var script = document.createElement("script"); script.src = scripts[i]; head.appendChild(script); } })();

    可以将上述代码写function 的外面就可以引入的。

    上述代码是在 fcpub.js 中引入 /fceform/js/fcvalid.js 的代码。你只要按这个样子写就行了

    把那段代码拷到userfunc.js文件中？里面fcvalid.js改成我自己写的js文件.

    如果是多个， [ fcpubdata.Path + "/fceform/js/fcvalid.js" , fcpubdata.Path + "/fceform/js/fcvalid.js"],中间加个“,”分开.

4. eform表单中的自定义函数能写那些东西
       在eform的自定义函数页面,所有js的东西都可以写,eform就是将自定义函数中写的所有东西外加上〈script〉 〈/script〉 后放到页面的body中的.所以在页面上的 <script>中写的东西都可以写.
