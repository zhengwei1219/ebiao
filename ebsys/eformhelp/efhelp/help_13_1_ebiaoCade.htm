1.列表表单用于打开显示e表控件的表单
        单击列表表单进入列表表单设计界面，列表表单用来选择一条记录在打开表单的e表控件中显示，它需要设置数据集，表格控件。数据集中要设置显示数据的sql语句，sql语句如下：

select FCS_EMPLOYEE.semployeeid,FCS_EMPLOYEE.semployeecode,FCS_EMPLOYEE.semployeename,'显示' as show from FCS_EMPLOYEE

添加了一个字段show（显示），用于查看报表内容，在数据集show字段的地址输入框中输入javascript:uf_href();void(0)，uf_href()函数定义如下：

function uf_href(){

         window.open('../../fceform/ereport/ebrun.htm?file=/ebfile/eformprint/eb_cardinput.htm&urlpara=yes&fcKeyValue='+dataset1.Field('semployeeid').Value);

}

       
2. 给双击列表表单表格控件中的列时，弹出的表单添加e表控件
2.1 在表单中添加e表控件并给e表控件装入报表文件
      单击 卡片表单进入卡片表单设计界面，在表单设计器中单击e表图标，在设计界面上弹出e表控件。双击e表控件弹出e表网页对话框，如果有现成的报表文件，就不用再设计报表文件，直接在e表网页对话框的页面中，单击打开文件图标，弹出报表文件选择页面，选中报表文件名单击【确定】按钮

即可装入报表文件，单击e表网页对话框中的【确定】按钮，

把报表文件保存到e表控件中。
2.2  给e表控件中添加控件
         单击e表控件，e表控件为选中状态，单击表单设计器的工具栏中的text控件图标，text控件被放入e表控件中，然后可以拖动text控件，拖入=ds1.职员编号的位置，安照此方法添加其它控件。

表单中要添加数据集控件设置数据集字段，提交方式，保存表名，保存主键类型，e表控件中放入的控件要和数据集绑定，在表单属性页中设置自定义工具栏。
