1.page页控件运行多个表单
    page页控件的每个子页都是url打开的运行界面,不能在表单设计器中给子页增加控件,它是直接链接到一个URL地址,在一个iframe元素中打开这个URL地址,每个子页都会有一个iframe元素.用户可以在page页控件的每个子页中打开一个表单,用它来实现子表单的功能,即一个表单中包含了其它的多个表单. 单击 page页控件运行多个表单 进入page页控件运行多个表单设计界面.
2.page页控件运行多个表单设置
    一个grid控件,一个page页控件 一个数据集控件.grid控件和数据集控件绑定,用于显示你需要看到的多条数据.page页控件用于装入你需要在page页控件中显示的文件.在数据集中的打开事件之后设置page页控件的第一个页签的url地址.在数据集的滚动之后设置page页控件的页签要打开的url.设置page页控件页签的url的方法是page控件id.SetUrl(page页控件的页签号,url地址);

page1.SetUrl(page1.selectedIndex,'../../fceform/common/djframe.htm?djsn='+url+'&djtype=FT&paravalue='+dataset1.Field('semployeeid').Value);

    在页签控件的单击事件中调用自定义函数uf_openUrl(),

    数据集的打开之后和移动记录之后函数中调用自定义函数uf_openUrl()

自定义函数内容如下

function uf_openUrl(){

         var url='help_4_7_page1';

         if(page1.selectedIndex == 1){url='help_4_7_page2';}

         page1.SetUrl(page1.selectedIndex,'../../fceform/common/djframe.htm?djsn='+url+'&djtype=FT&paravalue='+dataset1.Field('semployeeid').Value);

}

3.与其它视图对比
    它可以在一个page页控件中显示多个表单,其它视图都是一个表单.当前表单和page页控件中显示的子表单之间的参数传递是通过组装子表单的运行URL来实现的，即将要传的参数加到子表单的运行URL上。然后在子表单中再从URL上取下来。
