我国植树造林绿化统计
&nbsp;&nbsp;&nbsp;&nbsp; 再我国乃至世界植树造林，绿化环境已纳入重要课题，在我国西北方主要以抗旱树木为主，它在抗击风沙，防止沙漠扩大起着重要作用。在城市中它又起到了绿化环境，提高空气质量。面对西北地区雨量减少，沙漠化严重，从1997年到2006年5年累计增长了 80％。本文研究选取甘肃，宁夏，内蒙，北京，天津为例，进行统计分析，以了解我国植树绿化状况。
